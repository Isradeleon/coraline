Sidebar

A sidebar looks exactly like the one in this documentation page. The sidebar element will be fixed and 20% width in desktop and large screen devices. You should place your content inside an element with the sidebar-content class next to it.

As the sidebar is made to work with the navbar component for responsive functionality, add a simple navbar component as the first child of the sidebar-content element with the with-sidebar class. This navbar wont be responsive in the same way as the normal navbar, as it is designed to work as a toggler for the sidebar visibility on mobile devices.
